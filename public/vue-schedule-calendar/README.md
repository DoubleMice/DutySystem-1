# vue-schedule-calendar

还在 [demo](https://yscoder.github.io/vue-schedule-calendar/) 阶段。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
